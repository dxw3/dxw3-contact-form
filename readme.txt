== Description ==

Very simple contact form. Form to test dxw3's Form Bot Spam Block plugin protection of an ajax form.
Usage: add [dxw3-form] shortcode on the page.

== Installation ==

1. Upload the plugin zip file using the WordPress plugins -screen add-new-plugin -functionality.
2. Activate the plugin on the plugins -screen in WordPress

The plugin works automatically without configuration.
